# 🏪 Convenience Store Management System

편의점 운영을 위한 **데이터베이스 설계 및 관리 시스템**입니다.

SQLite 기반으로 구현되었으며, 상품·재고·판매·고객·발주 데이터를 통합 관리하고 다양한 통계 및 조회 기능을 제공합니다.

---

## 📌 프로젝트 목표

* 편의점 운영에 필요한 데이터베이스 설계
* 정규화된 관계형 데이터베이스 구축
* SQL을 활용한 데이터 조회 및 분석
* CLI 기반 관리 시스템 구현
* 더미 데이터 생성 및 시뮬레이션 환경 구축

---

## 🛠️ 기술 스택

* **Language** : Python
* **Database** : SQLite
* **Query** : SQL
* **Interface** : CLI (Command Line Interface)

---

## 🗄️ 주요 관리 기능

### 📦 상품 관리

* 상품 조회
* 카테고리 관리
* 브랜드 관리

### 🏪 매장 관리

* 전체 매장 조회
* 특정 매장 정보 조회

### 📊 재고 관리

* 매장별 재고 조회
* 안전재고 기준 부족 상품 조회
* 자동 발주 시뮬레이션 지원

### 💳 판매 관리

* 판매 내역 조회
* 매출 분석
* 상품별 판매량 분석

### 👥 고객 관리

* 회원(Member) 관리
* 비회원(Guest) 관리
* 고객 정보 조회

### 🚚 발주 및 배송 관리

* 발주 내역 조회
* 공급업체 관리
* 입고(Delivery) 관리

## 🎲 더미 데이터 생성

`generate_data.py`를 통해 테스트용 데이터를 자동 생성합니다.

생성 데이터 예시

* 매장 8개
* 상품 18개
* 고객 150명
* 판매 내역 600건
* 발주 내역 100건

```bash
python generate_data.py --reset
```
