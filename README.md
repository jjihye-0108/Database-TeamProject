# 🏪 편의점 관리 시스템 (Convenience Store Management System)

## 📌 1. 프로젝트 소개

본 프로젝트는 데이터베이스 과목 팀 프로젝트로, 편의점의 상품 판매, 재고 관리, 발주 및 입고 업무를 관리하기 위한 데이터베이스 시스템이다.

SQLite 데이터베이스를 기반으로 구현하였으며, Python CLI(Command Line Interface)를 통해 사용자가 데이터에 접근하고 다양한 업무를 수행할 수 있다.

선정한 유통업체는 **이마트24(Emart24)** 이며, 다음 기능을 제공한다.

* 상품 조회
* 매장 조회
* 재고 조회
* 구매 처리
* 발주 처리
* 입고 처리
* 자동 재주문 발주

---

## 🛠️ 2. 개발 환경

* Python 3.x
* SQLite3

## 📂 3. 프로젝트 구조

```text
DATABASE-TEAMPROJECT
│
├── README.md
├── convenience.db
```

### 📄 파일 설명

| 파일                     | 설명               |
| ---------------------- | ---------------- |
| schema.sql             | 데이터베이스 스키마 생성    |
| queries.sql            | 분석용 SQL 쿼리       |
| generate_dummy_data.py | 더미 데이터 생성        |
| main.py                | CLI 프로그램 시작      |
| product.py             | 상품 조회 기능         |
| inventory.py           | 재고 조회 기능         |
| store.py               | 매장 조회 기능         |
| transaction.py         | 구매, 발주, 입고 처리 기능 |
| convenience.db         | SQLite 데이터베이스    |

## 🗄️ 4. 데이터베이스 생성 방법

### 1) 스키마 생성

```bash
sqlite3 convenience.db < schema.sql
```

### 2) 더미 데이터 생성

```bash
python generate_dummy_data.py --reset
```

실행 시 다음 데이터가 생성된다.

* 🏪 매장
* 📦 상품
* 🏷️ 브랜드
* 🚚 공급업체
* 👤 고객(Member / Guest)
* 📊 재고
* 🛒 판매 내역
* 📑 발주 내역
* 📥 입고 내역

## ▶️ 5. 프로그램 실행 방법

CLI 폴더로 이동 후 실행한다.

```bash
cd cli
python main.py
```

## ⚙️ 6. 제공 기능

### 🛍️ 상품 조회

* 전체 상품 조회
* 상품명 검색
* 브랜드별 상품 조회

### 📦 재고 조회

* 매장별 재고 조회
* 재고 부족 상품 조회

### 🏪 매장 조회

* 전체 매장 조회
* 특정 매장 조회

### 💳 구매 처리

구매 시 다음 작업이 자동 수행된다.

* SALES 생성
* SALESDETAIL 생성
* INVENTORY 재고 차감

### 📑 발주 처리

발주 시 다음 작업이 수행된다.

* PURCHASEORDER 생성
* ORDERDETAIL 생성

### 🚚 입고 처리

입고 시 다음 작업이 수행된다.

* DELIVERY 생성
* INVENTORY 재고 증가
* PURCHASEORDER 상태 변경

### 🤖 자동 재주문 발주

안전재고 이하 상품을 탐지하여 자동으로 발주를 생성한다.

## 📈 7. 주요 분석 쿼리

프로젝트에는 다음과 같은 OLAP 성격의 분석 쿼리가 포함되어 있다.

1. 판매 실적 상위 5개 매장 조회
2. 각 매장별 판매량 TOP 20 제품 조회
3. 시·도별 판매량 TOP 20 제품 조회
4. 펩시가 코카콜라보다 많이 판매된 매장 수 조회
5. 우유와 함께 구매한 상품 TOP 3 조회
6. 재고 부족 상품 조회
7. 브랜드별 매출 순위 조회
8. 공급업체별 취급 브랜드 조회
9. 특정 카테고리 상품 조회
10. 여러 카테고리에 속한 상품 조회
